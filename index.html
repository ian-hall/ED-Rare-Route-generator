Elite: Dangerous Rare Routes

A program I put together to generate rare trade routes for the game Elite: Dangerous. Uses a genetic algorithm to throw a bunch of systems together until it gets a group of systems that isn't completely horrible. System data is based off the spreadsheet found here. The max route length is set at 35 systems because that is around where the program stops regularly finding solutions in a reasonable amount of time. You can find it here on GitHub.

If you don't know what the heck any of this means, you can go to my other page to make some fake rare items: Random Rares
